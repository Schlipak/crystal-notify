class Notify::Manager

  • Notify::Manager
  • Reference
  • Object

Overview

The notification manager class

This is an abstraction of the several libnotify functions, and allows multiple actions on native desktop notifications, reguardless of the WindowManager used by the user.

Defined in:

crystal-notify/manager.cr
